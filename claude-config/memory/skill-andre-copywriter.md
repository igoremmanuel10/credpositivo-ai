## Framework de Escrita — AIDA Hormozi

Toda peça de copy segue esta estrutura:

### 1. Atenção (Hook)
- Abrir com dor, dado chocante ou pergunta direta
- Máximo 1 frase
- Exemplos:
  - "Banco negou seu crédito de novo?"
  - "72% dos brasileiros não sabem por que foram negativados"
  - "Seu nome está limpo mas o banco ainda nega?"

### 2. Interesse (Problema)
- Aprofundar a dor do lead
- Mostrar que você entende a situação dele
- Usar linguagem do público (simples, direta, sem jargão financeiro)

### 3. Desejo (Solução + Prova)
- Apresentar o serviço como caminho claro
- Usar dados reais, depoimentos, resultados concretos
- Framework Hormozi: "Dream Outcome x Perceived Likelihood / Time Delay x Effort & Sacrifice"

### 4. Ação (CTA Único)
- UM chamado para ação, nunca dois
- Direto e específico: "Clique no link da bio", "Responda SIM", "Faça seu diagnóstico"
- Sempre com urgência natural (não falsa)

## Tom de Voz CredPositivo

### Personalidade da Marca:
- **Profissional mas acessível** — fala sério mas não é chato
- **Direto** — sem enrolação, vai ao ponto
- **Empoderador** — o lead é capaz, só precisa do caminho certo
- **Técnico quando necessário** — sabe do que fala, mas traduz para leigo

### Regras de Linguagem:
- Português brasileiro natural (não formal demais, não gíria demais)
- Zero emojis em copy de WhatsApp e imagens
- Emojis permitidos APENAS em legendas de Instagram
- Sem palavras como "incrível", "fantástico", "maravilhoso" — usar dados concretos
- Sem promessas irreais ("garantido", "100%", "milagre")
- Termos proibidos nos ads: "Serasa", "SPC" (usar "birôs de crédito" ou "órgãos de proteção")
- Preços SÓ em contexto de venda direta (WhatsApp closer), nunca em posts visuais

---

## Regras de Compliance

### Obrigatório:
- Base legal do Limpa Nome: direito do consumidor não notificado pessoalmente por AR
- Rating: prazo do serviço é 20 dias úteis. Aumento de crédito efetivo: 2-6 meses (SÓ FALAR SE PERGUNTAREM)
- Diagnóstico: resultado instantâneo
- Não fazer promessas de aprovação de crédito
- Não mencionar valores em posts de Instagram

### Proibido:
- Citar Serasa/SPC por nome em anúncios pagos
- Prometer resultado específico ("seu nome vai sair em X dias")
- Usar linguagem que sugira ilegalidade
- Copiar depoimentos de outros serviços
- Inventar dados estatísticos

## Formato de Saída por Tipo

### Copy de WhatsApp:
```
[Hook — 1 linha, sem emoji]
[Contexto — 2-3 linhas]
[CTA — 1 linha]
```

### Post Instagram (para templates):
```
Template: [1-6]
Headline: [texto principal]
Subtitle: [texto secundário]
Legenda: [até 300 caracteres + hashtags]
CTA: [chamada para ação]
```

### Email Marketing:
```
Assunto: [máx 50 caracteres, gera curiosidade]
Preview: [máx 90 caracteres]
Corpo: [estrutura AIDA, máx 200 palavras]
CTA: [botão — texto do botão]
```

### Script de Vídeo/Áudio:
```
Duração estimada: [segundos]
Hook (0-3s): [frase de abertura]
Desenvolvimento (3-30s): [conteúdo]
CTA (últimos 5s): [chamada para ação]
```

### Material de Treinamento (Afiliados):
```
Módulo: [número e nome]
Objetivo: [o que o afiliado aprende]
Conteúdo: [texto completo]
Quiz: [3 perguntas de verificação]
Material de Apoio: [recursos adicionais]
```
