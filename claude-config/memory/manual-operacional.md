# Manual Operacional — Como Usar o Time CredPositivo

## Regra #1: Você NÃO fala com todo mundo

Você é o dono. Fala com **3 pessoas no máximo** no dia a dia:

| Pessoa | Quando falar | Como chamar |
|--------|-------------|-------------|
| **Musk** | Decisão estratégica, direção, prioridades | "Musk, preciso decidir X" |
| **Jobs** | Execução operacional, cobrar resultado, resolver problema | "Jobs, status do funil" / "Jobs, preciso que X aconteça" |
| **Fernando** | Algo técnico quebrou, precisa de feature, deploy | "Fernando, precisa implementar X" |

Os outros 9 agentes são gerenciados pelo Jobs. Você NÃO precisa falar com eles diretamente (mas pode se quiser).

## Como Falar com Cada Agente

### Formato geral
Quanto mais direto, melhor. Sem enrolação.

### Musk (CEO) — Decisões
```
"Musk, [situação]. Qual a melhor decisão?"
"Musk, receita está em R$X. Meta é R$Y. O que priorizar?"
"Musk, tenho 3 opções: A, B, C. Qual seguir?"
```
Musk responde com: decisão + motivo + próxima ação + dono.

### Jobs (Orquestrador) — Execução
```
"Jobs, status geral"
"Jobs, preciso que [ação específica] aconteça"
"Jobs, lead [nome] está parado, resolve"
"Jobs, quero campanha de [produto], organiza o time"
```
Jobs responde com: quem vai fazer + prazo + como acompanhar.

### Fernando (Dev) — Técnico
```
"Fernando, implementa [feature]"
"Fernando, o bot está [problema]"
"Fernando, deploy da versão [X]"
```
Fernando responde com: o que vai fazer + prazo + checklist.

### Acesso direto (quando quiser, não obrigatório)
```
"Bia, cria 5 posts sobre [tema]"
"Andre, escreve copy de [tipo] para [produto]"
"Perola, cria landing page para [produto]"
"Rafael, como estão as campanhas?"
"Clara, como está o NPS?"
"Luan, relatório da semana"
```

## Rotina Sugerida

### Diária (5 min)
1. "Jobs, status geral" → ver se tem alertas
2. Verificar se Alex reportou problemas (WhatsApp ou #alex)

### Semanal (Segunda, 15 min)
1. "Musk, review da semana" → análise 80/20
2. "Luan, relatório semanal" → números do funil
3. "Jobs, prioridades da semana" → definir 3 foco
4. "Rafael, relatório de tráfego" → CPL, ROAS, criativos

### Mensal (1h)
1. "Musk, análise mensal completa" → direção estratégica
2. "Luan, comparativo mês anterior" → tendências
3. "Clara, NPS e upsell rate" → saúde do pós-venda
4. Decidir: o que escalar, o que matar, o que testar

## Cenários Comuns

### "Quero faturar mais este mês"
→ Musk diagnostica: Volume baixo? Conversão baixa? Ticket baixo?
→ Jobs executa a correção no ponto certo

### "Quero lançar campanha de Limpa Nome"
→ Jobs coordena:
  1. Andre cria copy
  2. Perola cria criativos
  3. Rafael sobe ads
  4. Augusto qualifica leads que chegam
  5. Paulo fecha

### "Cliente insatisfeito postou reclamação"
→ Jobs aciona Clara (<2h resposta)
→ Clara resolve + faz follow-up
→ Se necessário, Andre prepara comunicação

### "Bot não está respondendo"
→ Fernando direto (urgência técnica)
→ Alex auto-diagnostica se possível
→ Fernando corrige e faz deploy

### "Quero criar conteúdo pro mês"
→ Bia monta calendário editorial
→ Andre escreve as copies
→ Perola cria as artes
→ Rafael alinha pago + orgânico
