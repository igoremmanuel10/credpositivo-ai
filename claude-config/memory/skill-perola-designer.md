## Filosofia Anti-AI-Slop

### NUNCA usar:
- Inter, Roboto, Arial, system fonts genéricos
- Gradientes roxos em fundo branco
- Layouts previsíveis e cookie-cutter
- Cards idênticos com cantos arredondados uniformes

### SEMPRE:
- Direção estética ousada e intencional
- Tipografia como base (fontes únicas, pares refinados)
- Paleta dominante + acentos (não distribuir igualmente)
- Composição espacial inesperada (assimetria, overlap, grid-breaking)
- Backgrounds com atmosfera (mesh, texturas, noise, patterns)

## Workflow

### Antes de Criar
1. **Propósito**: Que problema resolve? Quem usa?
2. **Tom estético**: Minimal? Luxo? Editorial? Playful?
3. **Restrições**: Framework, device, performance
4. **Diferenciação**: O que torna INESQUECÍVEL?

### Durante
- Body text primeiro (fonte, tamanho, line-height, max-width)
- CSS variables para consistência
- Animações em momentos de alto impacto
- Mobile-first sempre

### Tipografia (regras permanentes)
- Aspas curvas, nunca retas
- En dash para ranges (1–10), em dash para quebras
- All caps: SEMPRE com letter-spacing 5-12%
- Max 2 fontes por projeto
- Line length: 45-90 chars (max-width: 65ch)
- Line height: 1.2-1.45

### QA Visual
1. Verificar typos e placeholders
2. Inspecionar overlaps, alinhamento, contraste
3. Testar responsividade (mobile, tablet, desktop)
4. Acessibilidade (contraste 4.5:1, focus visible, alt text)
5. "Como ficaria em um museu?"
