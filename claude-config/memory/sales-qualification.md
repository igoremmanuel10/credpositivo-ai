# Sales Qualification Skill

## Scope
Covers: Defining "qualified" (fit + need + access + urgency), disqualification rules, qualification scorecard, discovery/qualification scripts, stage exit criteria, rollout + measurement.

## When to Use
- Pipeline full but nothing closes
- Need lead qualification scorecard + disqualification criteria
- Discovery/qualification script for SDRs/AEs
- Stage exit criteria to prevent deal rot
- Spending time on bad-fit leads

## When NOT to Use
- No ICP hypothesis or pre-product/market fit
- Need full sales org design (use building-sales-team)
- Need pricing/packaging strategy
- Lead scraping/spammy outreach

## Inputs Required
- Product + target customer (1 sentence each)
- ICP hypothesis (industry, size, buyer titles, exclusions)
- Sales motion (inbound/outbound/PLG/enterprise) + current stages
- Typical deal profile (ACV, cycle length, stakeholders)
- Current pain: what "bad leads" look like (3-5 recent losses/stalls)
- Constraints: rep capacity, response times, tooling, handoffs

## Outputs (Sales Qualification Pack)
1. Context snapshot (ICP, motion, constraints, "what qualified means")
2. Qualification charter (ICP segments, disqualifiers, criteria, stage exit criteria)
3. Qualification scorecard (weighted criteria + thresholds + examples)
4. Discovery/qualification script (agenda, opener, question bank, disqualify talk track)
5. CRM artifacts (qualification notes template + required fields + pipeline hygiene)
6. Rollout + measurement plan (training, coaching, KPIs, iteration loop)
7. Risks / Open questions / Next steps

## Workflow (7 Steps)

### 1) Intake + Define Qualification Decision
- Clarify: pursue now, nurture, or disqualify
- Define unit of analysis (lead, account, opportunity)
- Define who qualifies (SDR, AE, founder)
- Output: Context snapshot + decision definitions
- Check: Rep can answer "What decision am I making after this call?"

### 2) Lock ICP Segments + Hard Disqualifiers
- Create 1-3 ICP segments (primary/secondary) + exclusions
- Write hard disqualifiers (fast "no" rules)
- Include graceful disqualify talk track
- Check: Disqualifiers observable within first 10-15 minutes

### 3) Build Qualification Scorecard (fit x need x access x urgency)
- Choose 5-8 criteria, weight them
- Clear scoring anchors (0/1/2/3 or 1-5)
- Thresholds for accept, nurture, reject
- "Minimum must-pass" criteria (non-negotiables)
- Check: Two reps scoring same deal land within ~1 tier

### 4) Define Stage Exit Criteria
- Write stage definitions and exit criteria (evidence required to advance)
- Pipeline hygiene: required fields, max stage age, "no next step, no stage"
- Stalled-deal policy
- Check: Every opportunity has (a) next step with date, (b) owner, (c) reason it can win

### 5) Create Discovery/Qualification Script
- 20-30 min call flow surfacing: current state, pain/impact, trigger/urgency, stakeholders, constraints, next step
- Include disqualify path and nurture path
- Check: Script produces accept/nurture/reject decision by end of call

### 6) CRM Note Template + Rollout Plan
- Structured notes template capturing scorecard inputs
- Required fields, lost/nurture reasons
- 2-week rollout plan (training, call reviews, calibration)
- Check: Manager can audit 10 opportunities quickly

### 7) Quality Gate + Finalize
- Always include: Risks, Open questions, Next steps
- Include "iteration loop" (revisit after 20-30 calls)

## Key Principles
- Qualification = fit + need + access + urgency
- "No next step, no stage" pipeline hygiene
- Hard disqualifiers protect rep time
- Two reps scoring same deal should agree on tier
- Always include graceful disqualify talk track
