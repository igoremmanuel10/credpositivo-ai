## Templates por Formato

### 1. Carrossel Educativo (Instagram)

**Estrutura (7-10 slides)**:

```
SLIDE 1 — CAPA (Gancho)
"[Numero] [coisas] que [resultado desejado]"
Ex: "5 motivos que fazem o banco negar seu credito"

SLIDE 2-8 — CONTEUDO
1 ideia por slide
Titulo curto + explicacao em 2-3 linhas
Icone ou numero grande

SLIDE 9 — RESUMO
Lista rapida dos pontos

SLIDE 10 — CTA
"Quer descobrir sua situacao real?"
"Diagnostico de Rating por R$97"
"Link na bio ↗"
```

**Legenda complementar**:
```
[Gancho emocional - 1 linha]

[Resumo do conteudo - 2-3 linhas]

[Pergunta de engajamento]

[CTA]
Faca seu Diagnostico → link na bio 📊

[Hashtags - em comentario separado]
```

### 2. Post Feed Unico

**Formula PAS (Problema-Agitacao-Solucao)**:
```
[VISUAL] Frase de impacto sobre o problema

[LEGENDA]
❌ [Problema que o publico vive]

[Agitacao - consequencias de nao resolver]

✅ [Solucao que a CredPositivo oferece]

[CTA] Primeiro passo: Diagnostico de Rating (R$97)
Link na bio 📊
```

**Formula AIDA (Atencao-Interesse-Desejo-Acao)**:
```
[VISUAL] Dado impactante ou pergunta provocativa

[LEGENDA]
📊 [Atencao - dado surpreendente]

[Interesse - explicacao relevante]

[Desejo - como resolver / beneficio]

[Acao - CTA claro]
```

### 3. Reel / TikTok Script

**Template Mito vs Verdade**:
```
HOOK (0-3s): "Voce acredita que [mito comum]?"
MITO (3-8s): Apresenta o mito com texto na tela
VERDADE (8-20s): Explica a verdade com autoridade
VIRADA (20-25s): "E sabe o que mais?"
CTA (25-30s): "Faz o Diagnostico pra descobrir sua situacao real"

LEGENDA: [Resumo + CTA + hashtags]
AUDIO: [Trend do momento ou audio original]
```

**Template Dica Rapida**:
```
HOOK (0-3s): "Dica que ninguem te conta sobre [tema]"
CONTEUDO (3-25s): Explica a dica de forma clara
CTA (25-30s): "Salva esse video e manda pra quem precisa"

LEGENDA: [Dica resumida + CTA + hashtags]
```

**Template Storytime**:
```
HOOK (0-3s): "Essa pessoa tava negativada ha [X] anos..."
HISTORIA (3-40s): Conta o caso de forma envolvente
RESULTADO (40-50s): "Hoje ela [resultado positivo]"
CTA (50-60s): "Quer o mesmo? Link na bio"
```

### 4. Stories (Sequencia)

**Template Educativo**:
```
STORY 1: Pergunta provocativa (fundo colorido)
STORY 2: "A maioria das pessoas nao sabe que..."
STORY 3: Explicacao com dado/fato
STORY 4: "E a boa noticia e..."
STORY 5: CTA com link/sticker de link
```

**Template Enquete**:
```
STORY 1: "Voce sabia que..." + ENQUETE (Sim/Nao)
STORY 2: Resultado/explicacao
STORY 3: CTA
```

**Template Caixinha de Perguntas**:
```
STORY 1: "Me pergunta qualquer coisa sobre credito/score/SPC"
STORY 2-5: Respostas as perguntas (conteudo gerado pelo publico)
STORY 6: CTA pro Diagnostico
```

### 5. Email / WhatsApp Broadcast

**Template Educativo**:
```
Oi [nome]! 👋

[Dado interessante sobre credito/score]

[Explicacao em 2-3 frases curtas]

[Dica pratica]

Quer descobrir exatamente como ta sua situacao?
→ Faz o Diagnostico de Rating: credpositivo.com/cadastro

Qualquer duvida, me chama aqui! 📱
```

**Template Follow-up**:
```
Oi [nome]! 😊

Lembra que a gente conversou sobre [tema]?

[Novidade/dica relacionada]

Se quiser dar o primeiro passo:
→ Diagnostico de Rating por R$97
→ credpositivo.com/cadastro

Abraco! ✅
```

## Estrutura de Blog / Artigo SEO

```
TITULO: [Keyword principal] — [Beneficio/Promessa]
Ex: "Como Limpar Seu Nome do SPC em 2026 — Guia Completo"

META DESCRIPTION: [155 chars max com keyword + CTA]

H1: [Titulo]

INTRO (100-150 palavras):
- Hook com dado/problema
- Promessa do artigo
- Keyword no primeiro paragrafo

H2: O que e [tema]
[Explicacao acessivel]

H2: Por que [tema] e importante
[Consequencias + motivacao]

H2: Como [resolver/fazer] — Passo a Passo
H3: Passo 1
H3: Passo 2
H3: Passo 3

H2: [Keyword] na pratica — Exemplo Real

H2: Perguntas Frequentes (FAQ)
[3-5 perguntas com respostas curtas — featured snippet]

H2: Conclusao
[Resumo + CTA pro Diagnostico]

CTA FINAL: Box destacado com link pro cadastro
```

## Formulas de Headline

### Para Carrosseis
- "[Numero] [coisas] que [resultado]"
- "Como [acao] em [prazo]"
- "Por que [problema acontece] (e como resolver)"
- "[Numero]% das pessoas [fazem errado]. Voce tambem?"

### Para Reels
- "Ninguem te conta isso sobre [tema]"
- "PARE de [erro comum]"
- "Se voce [situacao], assiste ate o final"
- "[Mito] vs [Verdade]"

### Para Stories
- "Voce sabia que...?"
- "Teste rapido: [pergunta]"
- "Me pergunta sobre [tema]"
- "A verdade sobre [tema]"
