# Skill: Bia Social Media — CredPositivo

Skill de criacao de conteudo de social media para a CredPositivo. Especializada em credito, financas pessoais e educacao financeira.

## Workflows

### 1. Criar Post Instagram
**Trigger**: "cria post", "post instagram", "post pro feed", "conteudo instagram"

**Steps**:
1. Identificar o tema/pilar (educativo, inspiracional, conversacional, promocional)
2. Consultar `references/content_frameworks.md` para o formato adequado
3. Criar visual sugerido + legenda + hashtags + CTA
4. Otimizar para o algoritmo do Instagram (consultar `references/social_media_optimization.md`)

### 2. Criar Carrossel
**Trigger**: "carrossel", "carousel", "slides"

**Steps**:
1. Definir tema e numero de slides (ideal: 7-10)
2. Criar estrutura: Capa (gancho) → Conteudo → CTA
3. Cada slide com 1 ideia clara, texto curto
4. Legenda complementar com CTA pro link na bio

### 3. Criar Reels/TikTok
**Trigger**: "reel", "reels", "tiktok", "video curto"

**Steps**:
1. Definir hook dos primeiros 3 segundos
2. Roteiro de 30-60 segundos
3. CTA final
4. Sugestao de audio/trend
5. Legenda com hashtags

### 4. Criar Stories
**Trigger**: "stories", "story", "sequencia de stories"

**Steps**:
1. Planejar sequencia (3-7 stories)
2. Incluir elementos interativos (enquete, caixinha, quiz)
3. Story final com CTA e link
4. Manter narrativa fluida

### 5. Calendario Editorial
**Trigger**: "calendario", "planejamento", "plano de conteudo"

**Steps**:
1. Consultar `assets/content_calendar_template.md`
2. Distribuir conteudo nos pilares (40/25/25/10)
3. Variar formatos (feed, carrossel, reel, story)
4. Incluir datas comemorativas e sazonalidade
5. Definir responsavel e status de cada peca

### 6. Copy para WhatsApp Broadcast
**Trigger**: "whatsapp", "broadcast", "mensagem em massa"

**Steps**:
1. Texto curto e direto (max 500 caracteres)
2. Emoji moderado
3. CTA claro pro Diagnostico ou link
4. Tom pessoal (como se fosse 1-a-1)

### 7. Campanha Completa
**Trigger**: "campanha", "lancamento", "campanha completa"

**Steps**:
1. Definir objetivo e produto foco
2. Criar pecas para cada canal (Instagram, TikTok, WhatsApp)
3. Sequencia temporal (aquecimento → lancamento → sustentacao)
4. Scripts de video + copies + visuais
5. Metricas de acompanhamento

### 8. Analise de Voz da Marca
**Trigger**: "analise de voz", "brand voice", "consistencia"

**Steps**:
1. Usar `scripts/brand_voice_analyzer.py` no conteudo
2. Comparar com guidelines em `references/brand_guidelines.md`
3. Sugerir ajustes para alinhar com a voz CredPositivo

### 9. Otimizacao SEO de Conteudo
**Trigger**: "seo", "otimizar", "palavras-chave"

**Steps**:
1. Usar `scripts/seo_optimizer.py` no conteudo
2. Analisar densidade de keywords, estrutura, legibilidade
3. Sugerir melhorias baseadas no score

## Arquivos de Referencia

- `references/brand_guidelines.md` — Voz da marca, tom, personalidade, linguagem
- `references/content_frameworks.md` — Templates e estruturas para cada formato
- `references/social_media_optimization.md` — Best practices por plataforma
- `assets/content_calendar_template.md` — Template de calendario editorial
- `scripts/brand_voice_analyzer.py` — Analise automatica de voz da marca
- `scripts/seo_optimizer.py` — Analise SEO de conteudo
